# zkVote Console

Privacy-first governance console built with Next.js, React, TypeScript, and Tailwind CSS.

## Local development

```sh
npm install
npm run dev -- -H 0.0.0.0 -p 3101
```

Open [http://localhost:3101](http://localhost:3101).

## Available scripts

```sh
npm run dev
npm run build
npm run start
npm test
npm run typecheck
```

## Tech stack

- Next.js Pages Router
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
